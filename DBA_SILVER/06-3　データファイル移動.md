# データファイル移動
## オフライン移動
① 移動対象のデータファイルをオフライン化
ALTER TABLE 表領域名 OFFLINE;
②データファイル移動
- mvなどで移動
③Oracleにデータふぁいるがいどうしたこ
- ALETR TABLE 表領域名 RENAME DATAFILEでデータファイルが移動したことを認識させる 
- ALTER TABLE 表領域名 ONLINE

## オンライン移動
- ALTER **DATABASE** MOVE DATAFILE [ファイルパス(ID指定可)] TO [ファイルパス]  [REUSE];
REUSE・・・既存ファイルが存在する場合
