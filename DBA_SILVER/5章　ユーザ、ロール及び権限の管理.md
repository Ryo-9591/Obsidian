# ユーザとユーザ作成

| SYSユーザ     | SYSTEMユーザ  |
| ---------- | ---------- |
| SYSDBA権限持ち | SYSDBA権限なし |
CREATE USER user01  
IDENTIFIED BY Password123
DEFAULT TABLESPACE tbso1 
TEMPORARY TABLESPACE temp •--  
-デフォルト一時表領域：temp  
QUOTA 10M ON tbs01  
クオータ：表領域tbs01に10Mバイト  
QUOTA UNLIMITED ON tbs02  
クオータ：表領域tbsO2に無制限  
PROFILE DEFAULT  
プロファイル：DEFAULT  
PASSWORD EXPIRE  
パスワード期限切れ  
ACCOUNT UNLOCK;  
アカウント：使用可能（アンロック）
