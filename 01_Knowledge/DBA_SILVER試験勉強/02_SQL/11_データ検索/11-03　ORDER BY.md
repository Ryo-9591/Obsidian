# ORDER BY
データの並べる

ソート基準列にNULLがある場合は、
昇順では末尾、降順では先頭にNULLを出力
## 昇順
| データ型 | 昇順の意味         | 備考                    |
| ---- | ------------- | --------------------- |
| 数字   | 小さい → 大きい     | 例: 1 → 10 → 100       |
| 日付   | 古い日付 → 新しい日付  | 例: 2020 → 2021 → 2022 |
| 文字列  | 辞書順（アルファベット順） | 例: 'A' → 'B' → 'Z'    |
